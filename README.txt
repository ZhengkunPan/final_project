Description:
The project will involve analyzing data related to hired-ride trips from Uber and NYC Yellow cab spanning from January 2009 through June 2015. The data will be joined with local historical weather data to answer specific questions using SQL queries and a Jupyter notebook. The data will first be downloaded and cleaned before being stored in a SQLite database. The project will also involve creating visualizations to aid in the analysis.
